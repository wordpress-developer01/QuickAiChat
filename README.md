
# QuickAI Chat (DEMO)

**About this app:**
QuickAI Chat is an AI-powered assistant platform that allows users to generate articles, blog titles, images, and perform resume reviews using advanced AI models. The app features user authentication (via Clerk), a dashboard, and a community section. It is designed as a demonstration of fullstack development with modern technologies and cloud integrations.

**Warning: This project runs in test mode!**

This project is a demo fullstack application built with React (Vite) and Node.js (Express), using Clerk for authentication, as well as integrations with AI and cloud services (Cloudinary, NeonDB, Gemini API, etc).

## Features
- Registration and login via Clerk (test keys)
- Article and blog title generation with AI
- Image creation and processing
- Resume review
- User community

## Important
- The project uses Clerk test keys (`pk_test_...`, `sk_test_...`), so:
  - There may be feature and user limits
  - Some features may be unavailable
  - Do not use for real data or production purposes

## How to run
1. Clone the repository
2. Install dependencies in both `client` and `server` folders
3. Fill in `.env` files with test keys (see `.env.example`)
4. Start the server and client

## Deployment
- For Render.com deployment, use Clerk test keys
- For full production use, obtain Clerk production keys
